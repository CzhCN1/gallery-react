# gallery-react
##[点击查看demo](http://czhcn.top/gallery-react/)  
一个基于React ES5的照片墙组件，并使用Webpack打包工具。

### 特点
1. 焦点图片居中，其余图片随机散布在周围
2. 点击居中图片翻转，显示背后的文字信息
3. 点击其他图片，该图片居中
4. 有索引列表，点击可切换至对应图片

### 安装
1. `git clone`至本地
2. `npm install` 安装所需依赖
3. `npm start` 在本地服务器运行

### 自定义
若要切换图片只需把新图片放在img文件夹下，修改`data -- imageDatas.json`
